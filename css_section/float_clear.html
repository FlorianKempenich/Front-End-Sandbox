The Most important!

Clearfix was used to fix a bug when two floated elements are stacked next to each other, see: ClearFix - CSS-Tricks

However, nowadays ClearFix is not necessary anymore. Flexbox and the CSS Grid solve all the layout problems Clear and Float were dealing with. And when it comes to simply floating text around an image Clear and Float work perfectly (I think).

Clearfix is only useful when the text in the paragraph is shorter than the floating block. But that shouldn't happen because if that's the case, it's simply better to use a Flexbox.
